# TuneWalrus

## Commands

- `Play <song title>`
  - Searches the song title on youtube and adds the first result to the queue
- `Skip`
  - Skips Song
- `Stop`
  - Stops play and deletes queue

## Installation

`npm install`

`ts-node --transpile-only src/server.ts`

still a few tiny TS things to be fixed, hence --transpile-only
